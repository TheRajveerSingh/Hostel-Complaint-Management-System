# Hostel Complaint Management System

A comprehensive, premium-designed web application for managing complaints in a hostel environment. Built with React and Vite, featuring multiple portals for Students, Staff, Wardens, and Supervisors.

## Features

- **Portals**: Distinct roles for students, maintenance staff, wardens, and supervisors.
- **Complaint Submission**: Easy-to-use interface for students to report issues (Carpentry, Housekeeping, Painting, etc.).
- **Real-time Tracking**: Monitor the status of complaints from submission to resolution.
- **Premium UI**: Modern tactical aesthetic using consistent design tokens and smooth animations.

## Tech Stack

- **Frontend**: React, Vite, Vanilla CSS
- **Design System**: Luminous Ledger
- **Branding**: HostelCare

## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/TheRajveerSingh/Hostel-Complaint-Management-System.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
